This app is a game ran on a python eventlet websocket server that implements the vanilla cryptography functions to ensure user data security.
The front-end of the app is written using html, css and vanilla js, the implementation was written with user ease and device compatability in mind.

To run the app:
1: Run Refresh_certs.py
2: install the CA-Cert.cer onto your machine
3: run GameServer.py
4: enjoy!
